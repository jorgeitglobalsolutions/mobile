```json
  "macros_por_100g": {
    "proteinas": 0.0,
    "carbohidratos": 0.0,
    "grasas": 0.0,
    "calorias": 0.0
  },
  "peso_por_unidad": 0.0,

}
```

### 7.2 UNIDADES DE MEDIDA

| Unidad | Uso |
|--------|-----|
| g (gramos) | Alimentos sólidos |
| ml (mililitros) | Líquidos |
| unidad | Alimentos contables (huevos, frutas) |
| cápsula | Suplementos |
| gotas | Propóleo |
| mg (miligramos) | Cafeína, vitaminas |
| cdita (cucharadita) | ~5ml |
| cdta (cucharada) | ~15ml |

### 7.3 RESTRICCIONES DIETÉTICAS

| Restricción | Descripción |
|-------------|-------------|
| gastritis | Apto para gastritis crónica |
| sin_gluten | Sin gluten / Celiaquia |
| sin_lactosa | Sin lácteos / Intolerancia lactosa |
| vegano | Sin productos animales |
| vegetariano | Sin carne ni pescado |
| sin_frutos_secos | Sin nueces, almendras, etc. |
